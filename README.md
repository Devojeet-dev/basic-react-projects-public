## Getting Started

Each project is a standalone Vite application. To run a project:

1. Navigate to the project directory:
   ```bash
   cd project-name
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
