This repository contains a collection of basic React projects created for the purpose of revising and strengthening React concepts.

## Projects

### 1. [Basic Counter](./1_basic-counter)
A simple counter application to understand state management (`useState`).

### 2. [Props and Tailwind CSS](./2_propsAndTailwindcss)
A project demonstrating the use of component props and styling with Tailwind CSS.

### 3. [(bg-changer) more of useState and tailwindcss](./3_bg-changer)
A project demonstrating the use of `useState` to change the background color of the page.

### 4. [Password Generator](./4_passwordGenerator)
A real-time password generator built with React and Tailwind CSS using `useState` and `useEffect`, featuring length control, number and special character toggles, and copy-to-clipboard functionality.

### 5. [Currency Converter](./5_currencyConverter/)
A real-time currency converter built with React and Tailwind CSS, leveraging a custom hook to fetch exchange rates and perform instant currency conversion with dynamic currency selection.

## Getting Started

Each project is a standalone Vite application. To run a project:

1. Navigate to the project directory:
   ```bash
   cd project-name
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
